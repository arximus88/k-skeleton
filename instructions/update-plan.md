# План оновлення проекту

## Зміни в структурі даних

### Параметри проекту
- Видалені параметри:
  - `disabled` (boolean)
  - `visible` (boolean)
  - `key` (string)
  - `sliderFrom` (number)
  - `sliderTo` (number)

- Нові параметри:
  - `live` (enum: "live" | "inactive" | "disabled")

### URL параметри
- Старі назви:
  - `projectUrl`
  - `clientUrl`
  - `iosUrl`
  - `androidUrl`
  - `url`

- Нові назви:
  - `url_project`
  - `url_client`
  - `url_ios`
  - `url_android`
  - `url_notion`
